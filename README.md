# Test Sepium — карточки товаров

Демонстрационный frontend-проект карточек товаров с интерактивным интерфейсом.

Проект реализует UI карточки товара с переключением изображений, навигацией и дополнительными элементами интерфейса. Основная задача — показать подход к верстке компонентов, организации интерфейса и работе с JavaScript.

## Стек

- HTML  
- CSS

- Fancybox  

---

## Реализованный функционал

- карточка товара с галереей изображений  
- переключение изображений (предыдущее / следующее)  
- навигация через кнопки  
- pagination для переключения изображений  
- отображение описания для каждого изображения  
- лайки товара  
- адаптивная верстка интерфейса  

---

## UI компоненты

В проекте реализованы интерфейсные элементы:

- **product card** — карточка товара  
- **image gallery** — галерея изображений  
- **navigation buttons** — кнопки переключения изображений  
- **pagination** — переключение изображений  
- **likes button** — кнопка лайков  

---

## Архитектура интерфейса

- компонентная структура карточки товара  
- разделение логики интерфейса через JavaScript  
- управление состоянием активного изображения  
- интерактивные элементы UI  

---

## Структура проекта

- /img — изображения проекта
- /css — стили
- /js — логика интерфейса
- index.html — основной файл страницы

---

## Особенности реализации

- переключение изображений внутри карточки товара
- синхронизация активного изображения и описания
- управление состоянием через JavaScript
- использование Fancybox для работы с изображениями
